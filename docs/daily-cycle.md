# Daily Cycle Guide

> Step-by-step walkthrough of the 6-phase daily cycle.

## The 6 Phases

1. **Plan** — Create PLANO.md with waves, tasks, and dependencies
2. **Approve** — Human/lead reviews and signs off
3. **Delegate** — Assign tasks to agents via Slack
4. **Execute** — Agents run their tasks with designated engines
5. **Audit** — Cross-check every completed task
6. **Report** — Log results, update index, commit
