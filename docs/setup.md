# Setup Guide

> This guide covers the initial setup for using the Agent Ops Workflow.

## Prerequisites

- [Hermes Agent](https://hermes-agent.nousresearch.com) installed and configured
- bash >= 4 (macOS / Linux)
- git
- (Optional) Slack workspace for agent communication

## Quick Setup

1. Run `./scripts/setup-workflow.sh ~/my-project "{{TEAM_NAME}}" "{{PROJECT_NAME}}"`
2. Customize templates in `planejamento-diario/TEMPLATES/`
3. Configure your agents and engines
4. Start your first daily cycle
