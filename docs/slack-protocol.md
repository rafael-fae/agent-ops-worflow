# Slack Protocol

> Structured agent communication patterns for multi-agent teams using Slack.

## Core Rules

- **Mention protocol**: Always use `<@USER_ID>` for agent mentions
- **One agent per mention**: Only the mentioned agent responds
- **Silence rule**: If not mentioned, stay silent
- **Lockdown**: "sinal vermelho" freezes all agents

## Channel Structure

- Dedicated channel per operation
- Orchestrator mentions agents with task IDs
- Agents report back in thread
