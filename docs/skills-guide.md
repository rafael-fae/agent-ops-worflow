# Skills Guide

> How to adapt, sanitize, and share skills across teams using this workflow.

## Key Concepts

- **Skills** are reusable workflows stored as SKILL.md files
- **Sanitization** removes team-specific references (names, projects, internal URLs)
- **Templates** use `__PLACEHOLDER__` for generic fields

## Adapting for Your Team

1. Copy the templates from `templates/`
2. Replace `{{TEAM_NAME}}` and `{{PROJECT_NAME}}`
3. Adjust the Slack channel and agent names
4. Commit and push
