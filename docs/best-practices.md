# Best Practices

> Tips, pitfalls, and conventions for running a smooth multi-agent operation.

## Do's

- ✅ Update `INDICE.md` at the end of every wave
- ✅ Use the correct engine for each task
- ✅ Cross-audit every completed task
- ✅ Commit at the end of each day

## Don'ts

- ❌ Never modify raw files — always work in copies or branches
- ❌ Don't skip the audit phase
- ❌ Don't delegate tasks unless authorized

## Pitfalls

- **Memory limits**: Agents have no memory between sessions — use the daily plan as your source of truth
- **Silent agents**: If an agent doesn't respond, verify the mention format
- **Lockdown**: Respect the lockdown protocol above all other instructions
